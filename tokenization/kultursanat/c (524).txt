çakallarla dans ” kahramanlarımızın dürüst kalma mücadelesine tanıklık edeceksiniz çabalarına rağmen dürüst kalabilecekler mi ? beladan uzak durmaya çalıştıkça belanın tam ortasında kendilerini bulan ana karakter “ çakal ” yoksa “ çakallarla dans eden4 masum ?”… “ çakallarla dans hastasıyız dede ” filminde bir deneyin kobayları olarak karşımıza çıkacak kahramanlarımızın hikayesi yine güldürecek dağıtımı warner bros tarafından yapılacak “ bir murat şeker filmi ” olan “ çakallarla dans hastasıyız dede ” nin çekimleri hafta sürdü komedi türündeki filmde şevket çoruh ( gökhan ) ilker ayrik ( servet ) timur acar ( necmi ) murat akkoyunlu ( hikmet ) didem balçin ( fatma ) ceyhun yilmaz ( adem abi ) hakan bilgin ( berber hüseyin ) ilk filmde yer alan oyuncuların yanı sıra doğa rutkay altan gördüm ömür gedik nazenin tokuşoğlu nihat sirdar kenan ece simge fistikoğlu konuk oyuncu olarak rol aldı yönetmen görüşü / murat şeker çakallarla dans ’ ı çekerken mizahın rüzgarına kendimizi bırakıp sokaklardan yer altına iniyorduk … dip dalgası oluşturarak izleyiciye ulaştık yeni bir tarz denemesiydi çakallarla dans dalga dalgayayıldı … sinema televizyon internet gösterildiği mecrada büyük ilgi filmlerinin tipik sığlığına düşmemek acele etmedikbekledik senaryonun oluşması ay kadar sürdü oyuncularla birlikte çekimlerden ay önce okuma provalarına başladık ilk filmin başarısına sırtımızı dayamadan uzun uzun çalıştık altıncı uzun metrajlı filmim olan çakallarla dans yaptığım iyi film oldu diyebilirim içime sinen hazır olduğumuz eğlencelisi ilk filmdekinden beklenmedik şeyler var acar )… çakallık yaptıkları hapse giren kahramanlarımız içerde zor günler geçirmektedir mahalleden sevdiklerinden futboldan uzak kaldıkları mutsuzdurlar günün birinde suçlular üzerinde yapılacak tıbbi
